## **Arrays**

### Operation Times

- **Access Value**: O(1)
- **Set Value**: O(1) 
- **Insert Value**:
    - At beginning of array: O(n)
    - At middle of array: O(n)
    - At end of array: O(n) for static arrays, amortized O(1) for dynamic arrays
- **Remove Value**:
    - At beginning of array: O(n)
    - At middle of array: O(n)
    - At end of array: O(1)
- **Copy Array**: O(n)
